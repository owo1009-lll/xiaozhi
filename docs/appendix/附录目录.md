# 附录目录

本文件夹用于集中存放论文附录所需表格与摘要文件。

包含文件：

- `附录A1-24个知识点完整定义表.csv`：24 个知识点完整定义表（ID、名称、子概念、对应题型、三级难度）
- `附录A2-BIMMS36题完整量表.csv`：BIMMS 36 题完整量表（英文 + 中文，标注反向题）
- `附录A3-TAM13题完整量表.csv`：TAM 13 题完整量表（英文 + 中文）
- `附录A4-MTE试题蓝图表.csv`：MTE 试题蓝图表（每课时分配题数 + 难度分布，不含具体题目和答案）
- `附录A5-MTE项目分析摘要表.csv`：MTE 项目分析摘要表（50 题的难度系数、区分度、KR-20 汇总）
- `附录总表.xlsx`：附录总表，整合以上全部内容为一个 Excel 文件
